# Project of comparing efficiency of cases
in this project, measurements were made for the following scenarios

## Comparing the speed of creating a list using:

- list comprehensions (list comprehensions)
- of the usual for loops
- of the numpy library
## Comparing the speed of data serialization and deserialization
- pickle
- json
- msgpack
In comparison size of the object: 1KB, 1MB, 10MB

## Comparing the string concatenation rate via += and .join

# Quick start

```make test```
